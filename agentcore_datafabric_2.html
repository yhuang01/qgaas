About Amazon Bedrock AgentCore

Amazon Bedrock AgentCore is a modular set of capabilities to build, deploy and operate production-grade agents securely and scalably using any framework and model.

AgentCore capabilities are optimized to work together or individually, giving you the flexibility to integrate with any open-source frameworks.

Enterprise Fabric
Use Amazon Bedrock AgentCore Enterprise Fabric to connect agents to your entire enterprise data ecosystem — on-premises, hybrid, and multi-cloud — without moving or replicating data.
How it works

Connect your data ecosystem

Enterprise Fabric automatically deploys the Teradata Teradata Fabric engine into your AWS account and registers your on-premises databases, cloud warehouses, and hybrid data sources as connectors — no data movement required.

Knowledge base synthesized automatically

Once connectors are registered, Enterprise Fabric automatically ingests schema metadata from all connected engines and synthesizes a Bedrock Knowledge Base — giving agents accurate, governed context for natural language to SQL generation.

Agents query any data, anywhere

Bedrock agents query on-premises, cloud, and hybrid data in a single federated call via the Enterprise Fabric gateway — with intelligent pushdown, row-level governance, and full audit logging. Data never leaves its security boundary.
